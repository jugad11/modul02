XAMPP

XAMPP adalah singkatan dari "X (dibaca sebagai 'cross')-Platform Apache MySQL PHP Perl", sebuah paket perangkat lunak yang memungkinkan Anda menjalankan server web lokal di komputer Anda. XAMPP mencakup komponen utama yang diperlukan untuk menjalankan aplikasi web dinamis, termasuk server web Apache, basis data MySQL, serta bahasa pemrograman PHP dan Perl.
